# Smart-Fitness-Trainer
An intelligent web-based fitness assistant that provides personalized workout recommendations, tracks user progress, and promotes a healthier lifestyle using AI-driven insights.
The AI Smart Fitness Trainer is designed to act as a virtual fitness assistant. Users can input their details (age, weight, goals), and the system generates customized workout suggestions using intelligent logic.
This project demonstrates the integration of FastAPI backend with a simple frontend UI, making it lightweight, fast, and easy to use.
-----> Features <-----
-> Personalized Workout Plans
Generates workouts based on user input like fitness goals, body metrics, and experience level
-> Fitness Tracking
Calculates and tracks basic metrics such as BMI and progress
-> Health Tips
Provides general suggestions for maintaining a healthy lifestyle
-> FastAPI Backend
High-performance backend for handling requests efficiently
-> Simple & Clean UI
Easy-to-use interface for smooth user experience
-> Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: FastAPI (Python)
Server: Uvicorn
AI Logic: Rule-based recommendation system
